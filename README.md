"Python Mini Projects Collection."

This repository contains two beginner‑friendly Python projects:

1. Vigenere Cipher Encryptor/Decryptor

A simple implementation of the classic Vigenere cipher algorithm for
text encryption and decryption using a predefined key.

Features

-   Encrypts plaintext using the Vigenere cipher
-   Decrypts back to original text
-   Handles spaces between words

------------------------------------------------------------------------
2. Expense Tracker (CLI)

A command‑line based expense tracker to manage and review your expenses.

Features

-   Add expenses with amount and category
-   View all expenses
-   Calculate total expenses
-   Filter expenses by category
